# Shopify-Stock-Checker
**Checks the stock and gets variants of shopify links!**

**Helpful for hyped releases!**
#Dependencies:

-Python 2.7

-BeautifulSoup

-Requests

Google how to get them

#How to use:
1. Open terminal.
2. Run script as:```python shopifychecker.py``` in terminal.
3. Paste shopify links.
